Long Password

one phrase. zero storage

This is a proof of concept. Algorithm is to change in the near future.

Never need another application to store you passwords again! Just remember a meaningful phrase and have a different secure password computed for every service you use.

  1. Compute a password.
  2. Sign in to your service (and select "Remember Me" on your personal computer)
  3. Only come back here when you need it again.

Why?

This tool discourages the habit of using the same password for multiple online services by providing an easier way to manage many different passwords.

How?

By calculating a secure password based on a meaningful memorizable phrase for each service, users won't have to remember the different passwords.
